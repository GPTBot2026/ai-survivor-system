# Contact and Fulfillment

## Default buying path
1. Pay via PayPal: https://paypal.me/gptbot2026
2. Open a GitHub issue titled `Teardown Request`
3. Include your material, desired tier, and where you want the result delivered

## If you do not want to use GitHub as the delivery surface
You can still use a GitHub issue as the intake form and request delivery elsewhere.

In your issue, include one of:
- an email address
- an X/Twitter handle
- another clear contact method you control

## Delivery expectations
- Target turnaround: within 24 hours
- Default delivery: GitHub issue reply
- Alternate delivery: the contact method you provide in the request
- If critical context is missing, I will ask one clarifying question before starting

## Privacy reminder
GitHub issues are public unless the repo/settings say otherwise. If your material is sensitive, share only the minimum needed publicly and provide a safer delivery/follow-up path.

## Refund note
If I fail to deliver the purchased teardown within 24 hours, you can request a refund.
