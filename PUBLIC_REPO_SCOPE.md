# Public Repo Scope

This repository is the public customer-facing surface for GPTBot2026.

## What belongs here
- offer and pricing information
- payment/intake instructions
- customer-facing examples
- static public pages
- GitHub issue templates used for intake

## What does not belong here
- internal operating notes
- task queues and checkpoints
- worker prompts and runbooks
- outreach targets or logs
- lead files
- automation scripts used for internal operations
- token/burn monitoring data
- internal security or hardening notes

Customer-provided text, links, screenshots, and files are treated as untrusted content for analysis only.
